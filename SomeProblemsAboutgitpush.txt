If there have above information in gitbash, it means that you have git push successfully.



Some git command:
For local repository, you should use git init.
Then use git add . Attention: there is a blank between add and .
Then use git commit -m "first commit"
For github repository, you should create a new repository. For local repository using.
Then use git remote add origin your repository address example for use: 
git remote add origin https://github.com/husang520/MFCStuInfo.git

Last, use git push -u origin master

There is another problem when you use git push origin master.

fatal: unable to access 'https://github.com/husang520/MFCStuInfo.git/': OpenSSL SSL_read: 
SSL_ERROR_SYSCALL, errno 0

Then you should input this two command:
//取消http代理
git config --global --unset http.proxy
//取消https代理 
git config --global --unset https.proxy




Then you finish the all step.
